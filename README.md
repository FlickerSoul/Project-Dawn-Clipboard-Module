# Project-Dawn-Clipboard-Module
## 禁止作为商用

Project Dawn 剪贴板查词模块

## Features

* 附带带本地韦氏词典（从Andriod上截取下来的数据库，9w词族，未在repo里提供）
* 在线金山词霸查词API
* 可在线发音并且缓存至本地目录
* CTRL+Alt+Shift+O 唤起和最小化
* ESC最小化
* CTRL+Shift+E/C/S 切换不同tab
* 软件内双击+右击或点击超链接查词
* Kindle 电脑版查词
* 可以切换Always On Top，是否复制查词，是否自动发音， 和是否使用Kindle模式。
